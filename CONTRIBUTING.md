# Contributing

Thank you for considering a contribution to this simple Ruby calculator project!

Guidelines
- Fork the repository and create a feature branch: `git checkout -b my-feature`.
- Follow the existing code style and keep changes small and focused.
- Add tests for new features using Minitest and ensure they pass locally.
- Run tests with:

```powershell
ruby test_calculator.rb
```

- Open a pull request describing your change; reference any related issues.

Code of Conduct
- Be respectful and constructive in discussions and reviews.

Thank you!

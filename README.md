# Ruby Calculator

Simple command-line calculator in Ruby.

Usage:

- Run a single expression:

```
ruby calculator.rb "2 + 3 * (4 - 1)"
```

- Start REPL:

```
ruby calculator.rb
```

Type `exit`, `quit`, or `q` to leave the REPL.

Notes:
- The calculator validates expressions to allow only digits, spaces and operators `+ - * / ( ) . ^`. The `^` is treated as exponentiation.
